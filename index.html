🕵️ Stealth Benchmark

Simulates detection techniques used by coding interview platforms (CoderPad, HackerRank, Codility…). Run this before your interview to make sure nothing gets flagged. · Built by blind.codes · GitHub

How to use
01 — Open this page in the same browser you will use for your interview.
02 — Set up your environment exactly as you would during the interview — overlay tools, second monitor, etc.
03 — Click ▶ Run all checks. Each check simulates a detection technique used by interview platforms.
04 — Watch the event log — it captures everything in real time: tab switches, focus loss, DevTools opening.
05 — Aim for 100%. A ✗ FAIL means a platform could flag you. A ~ INFO is informational only.
